Projects

Listed below is my collection of side projects that I've participated in.

Discord

I built an interactive bot, written in Python, intended for use on the application Discord. The bot joins a channel of its choice and can interact with guests by playing Hangman, Magic 8 Ball, Guessing Game, and more. It also gathers random images from Google and posts them in the channel by request.

Blue Lobster

I helped write a Reddit bot in Python. It processes Reddit's comment database, formatted by SQLite, and then learns how to create original content from such by using recurrent neural networks and more specifically a Sequence-to-Sequence model, given both the TensorFlow and PRAW API.

AudioCrawler

AudioCrawler is a ML application that takes music data (through spectroscopy graphs) and genres as input in order to classify future song inputs based on genre. I worked on the back-end with Keras, Python, and TensorFlow, as well as collected music data.

VacationPlanner

In De Anza College's Data Abstraction and Structures, I participated in a group project that demonstrated the class's knowledge of hash tables, binary search trees, and stacks. In C++, I worked on the hash table, user and file I/O, managers (insert, delete, list, search), and overall design.
